## For housing rental 

1. Goto INE.es
2. Select IPC en 1 clic
3. Select IPC en INEbase
4. Select Indices nacionales de subclases
5. Select 04110 or Alquiler de vivienda principal
6. Choose variacion anual, period
7. Click button CONSULTAR SELECCION
8. Print table
